#  University Library Management System


A simple Java-based console application for managing university library operations. This system allows librarians to manage students, books, and track book lending activities efficiently.

---

##  Features

-  **Add Students** - Register new students with ID, first name, and last name
-  **Add Books** - Add new books with ID, title, and publication year
-  **Lend Books** - Lend books to students with validation
-  **Display Info** - View student details and their borrowed books
-  **Data Validation** - Input validation for all fields
-  **Prevent Duplicate Lending** - Students cannot borrow the same book twice

---

##  Technologies Used

- **Java** - Core programming language
- **Object-Oriented Programming (OOP)** - Classes, Inheritance, Encapsulation
- **ArrayList** - Dynamic data storage
- **Scanner** - User input handling
